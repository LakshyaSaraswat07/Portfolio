## ✨ Features

- 🌙 Modern dark glassmorphism UI
- ⚡ React + Vite frontend
- 🧠 Smooth animations with Framer Motion
- 📱 Fully responsive design
- 📬 Contact form with Express, MongoDB, and Nodemailer
- 🚀 Single-service Render deployment

## 🛠️ Tech Stack

**Frontend:** React, Vite, CSS3, Framer Motion, React Icons, Axios  
**Backend:** Node.js, Express.js, MongoDB, Mongoose, Nodemailer  
**Deployment:** Render + MongoDB Atlas

## ▶️ Run Locally

```bash
git clone https://github.com/LakshyaSaraswat07/Portfolio.git
cd Portfolio
npm install --prefix frontend
npm install --prefix backend
npm run dev --prefix backend
npm run dev --prefix frontend
```
